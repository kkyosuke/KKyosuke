# ドキュメント (日本語)

本ディレクトリには、プロジェクトの開発プロセスや規約に関する日本語のドキュメントが格納されています。

## 目次

### 開発プロセス・規約 (`code/`)

- **[コミット規約](code/commit.md)**
  - Gitのコミットメッセージのフォーマット、変更の種類（type）、およびコミットの粒度に関するガイドライン。
- **[Issue作成方針](code/issue.md)**
  - 課題（Issue）作成時の粒度、構成、およびバグ報告や機能追加時のテンプレート。
- **[Pull Request作成方針](code/pull-request.md)**
  - プルリクエスト（PR）作成時の粒度、構成、テスト・エビデンスの記載、およびレビュー依頼時の注意点。
- **[コードレビュー観点](code/review.md)**
  - コードレビュー時のチェックリスト、PR記載内容の確認項目、およびレビューコメントのプレフィックス。
